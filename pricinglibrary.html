Pricing Library - Automated Tariff and Price Calculation

This is a C# DLL library designed to handle complex pricing rules. It loads pricing tables and tariffs from structured files, validates input parameters, calculates item prices, and applies volume or container discounts automatically.


  • Load tables and tariffs from structured pricing files
  • Validate input parameters for accuracy and consistency
  • Calculate current prices based on weight, format, zone, and mail type
  • Apply container and volume discounts automatically
  • Support for minimum quantities, average tray fill, and selection rules
  • Return standardized responses and detailed pricing breakdown
  • Can be integrated in any C# project as a DLL
